#### GRIND project :
- Server in Golang
- Client Reactjs
- Graphql And Database MongoDb
- Redis
- Pub/Sub Redis 
- Clean architecture 

### Install project:
#### Install Api:
- Install Golang [Here](https://golang.org/) if not installed
- Add conf your database Mongodb 
- go build
- go run main.go
